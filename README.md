# Shared Expense Tracker

A mobile-friendly, shared expense tracking application built for Abhishek and Stuti.

## Features
- **Shared Data**: Syncs seamlessly with Google Sheets.
- **Mobile First**: Designed to look and feel like a native app on your phone.
- **Dashboard**: Visual breakdown of expenses by category and budget status.
- **Offline-First UI**: Optimistic UI updates for a snappy experience.

## Setup
1.  Open `index.html` in your browser.
2.  Start tracking!

## Deployment
Hosted on GitHub Pages for easy mobile access.
